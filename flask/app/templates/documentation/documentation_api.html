Services

  • Kinase protein sequences (.fasta). Eg: 'P31749'
  • Kinase DNA sequences (.fasta). Eg: 'P31749'
  • Kinase information (.json). Eg: 'P31749'
  • Inhibitors information (.json). Eg: 'Abemaciclib'

* For the kinases information, the three services need the Uniprot Accession ID

Kinases Json service

Modify the parameter 'columns' in the url ({{url_for('kinase.kinase_json', kin_name = 'UNIPROT_ID', columns='REQUIRED_INFORMATION')}}) to get the required information:

  • 'all': returns all the information listed bellow
  • 'info': returns chromosome, Ensembl ID, Kinase Family, Full Protein Name, Genomic Sequence, Protein Sequence, Genomic location (start and end), Mass(Da), Gene name and Uniprot ID
  • 'isoforms': returns the list of isoforms, and , if possible, also the protein sequence
  • 'inhibitors': returns the Phase, Molecular Weight (mw), InchiKey, inhibitor name (inn_name) and the canonical Smiles
  • 'reactions': returns the list of reactions
  • 'cell_loc': returns a list of subcellular locations in which the protein has been registered
  • 'diseases': returns the disease description, name and effect
  • 'phosphosites_self': returns the genomic location (begin, end), residue position, modification type, and if aviablable, the kinases that target that position
  • 'phosphosites_targets': returns the modified residue (residue and position), the substrate gene, substrate accession Uniprot ID, and a window of +- 7 residues

Egs:

  • {{url_for('kinase.fasta_protein',kin_name ='P31749' )}}
  • {{url_for('kinase.fasta_gene',kin_name ='P31749' )}}
  • {{url_for('kinase.kinase_json', kin_name = 'P31749', columns ="all" )}}
  • {{url_for('kinase.kinase_json', kin_name = 'P31749', columns ="diseases" )}}
  • {{url_for('kinase.kinase_json', kin_name = 'P31749', columns ='')}}inhibitors,cell_loc,inhibitors

Inhibitors Json service

Modify the parameter 'columns' in the url ({{url_for('inhibitor.inhibitor_json', inhib_name = 'NAME_OF_INHIBITOR', columns='REQUIRED_INFORMATION')}}) to get the required information:

  • 'all': returns all the information listed bellow
  • 'info': returns the inhibitor name, canonical structure, inchikey, molecular weight (Da), and phase
  • 'targets': returns a list of target kinases and its uniprot ids
  • 'synonyms': returns synonyms for that particular inhibitor
  • 'pbid': Returns the Protein Databank ID for information related to the interaction between the inhibitor and a Kinase
  • 'families': returns a list of subcellular locations in which the protein has been registered

Egs:

  • {{url_for('inhibitor.inhibitor_json', inhib_name = 'abemaciclib', columns='all')}}
  • {{url_for('inhibitor.inhibitor_json', inhib_name = 'abemaciclib', columns='info')}}
  • {{url_for('inhibitor.inhibitor_json', inhib_name = 'abemaciclib', columns='')}}info,synonyms,families
