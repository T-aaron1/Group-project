Services

  • Kinase protein sequences (.fasta). Eg: 'P31749'
  • Kinase DNA sequences (.fasta). Eg: 'P31749'
  • Kinase information (.json). Eg: 'P31749'
  • Inhibitors information (.json). Eg: 'Abemaciclib'

* A valid Uniprot Accession ID is required regarding all Kinase information in order to use our services

Kinases Json service

(/kinase/UNIPROT_ID.json?columns=REQUIRED_INFORMATION) must be modified to ensure the required information is obtained:

  • 'all': returns all available information listed below
  • 'info': returns chromosome, Ensembl ID, Kinase Family, Full Protein Name, Genomic sequence, Protein sequence, Genomic location (start and end), Mass (Da), Gene name and Uniprot ID
  • 'inhibitors': returns the Phase, Molecular Weight (mw), InchiKey, inhibitor name (inn_name) and the canonical Smiles
  • 'isoforms':returns the list of isoforms and if applicable the subsequent protein sequence
  • 'reactions': returns a list of all possible kinase reactions
  • 'cell_loc': returns a list of all subcellular locations corresponding to the registered protein
  • 'diseases': returns the disease description, name and effect
  • 'phosphosites_self':returns the genomic location (start, end), residue position, modification type and if applicable, kinase target site/position
  • 'phosphosites_targets': returns the modified residue (residue and position), the substrate gene, substrate accession Uniprot ID and a window of +- 7 residues

Egs:

  • {{url_for('kinase.fasta_protein',kin_name ='P31749' )}}
  • {{url_for('kinase.fasta_gene',kin_name ='P31749' )}}
  • {{url_for('kinase.kinase_json', kin_name = 'P31749', columns ="all" )}}
  • {{url_for('kinase.kinase_json', kin_name = 'P31749', columns ="diseases" )}}
  • {{url_for('kinase.kinase_json', kin_name = 'P31749', columns ='')}}inhibitors,cell_loc,phosphosites_self

Inhibitors Json service

(/inhibitor/NAME_OF_INHIBITOR.json?columns=REQUIRED_INFORMATION) must be modified to ensure the required information is obtained

  • 'all': returns all the information listed below
  • 'info': returns the inhibitor name, canonical structure, inchikey, molecular weight (Da), and drug phase
  • 'targets': returns a list of target kinases and its Uniprot IDs
  • 'synonyms': returns synonyms specific to its inhibitor
  • 'pdbid': returns the Protein Databank ID which identifies the interaction between the inhibitor and a kinase
  • 'families': returns a list of subcellular locations corresponding to the registered protein

Egs:

  • {{url_for('inhibitor.inhibitor_json', inhib_name = 'abemaciclib', columns='all')}}
  • {{url_for('inhibitor.inhibitor_json', inhib_name = 'abemaciclib', columns='info')}}
  • {{url_for('inhibitor.inhibitor_json', inhib_name = 'abemaciclib', columns='')}}info,synonyms,families
