Services

  • Kinase protein sequences (.fasta), eg: 'P31749'
  • Kinase DNA sequences (.fasta), eg: 'P31749'
  • Kinase information (.json) 'P31749'

The three services need the Uniprot Accession ID

Json service

For the Json service, use the following paramenters to get the required information:

  • 'all': returns all the information listed bellow
  • 'info': returns chromosome, Ensembl ID, Kinase Family, Full Protein Name, Genomic Sequence, Protein Sequence, Genomic location (start and end), Mass(Da), Gene name and Uniprot ID
  • 'isoforms': returns the list of isoforms, and , if possible, also the protein sequence
  • 'inhibitors': returns the Phase, Molecular Weight (mw), InchiKey, inhibitor name (inn_name) and the canonical Smiles
  • 'reactions': returns the list of reactions
  • 'cell_loc': returns a list of subcellular locations in which the protein has been registered
  • 'diseases': returns the disease description, name and effect
  • 'phosphosites_self': returns the genomic location (begin, end), residue position, modification type, and if aviablable, the kinases that target that position
  • 'phosphosites_targets': returns the modified residue (residue and position), the substrate gene, substrate accession Uniprot ID, and a window of +- 7 residues
